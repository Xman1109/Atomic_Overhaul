    - Mod complexity selection introduced. Full (default) represents actual features when more is better. Simplified is a comfortable

      Optional Breeding feature replaces unrealistic Kovarex enrichment for Simplified mode. by @garrotte

    - (K2) Fixed a bug where nukes could only be fired by normal rocket launchers
    - (K2) Readjusted the graphite fuel value
    - prototype code for the fuel rods
    - some russian translations
    - made some changes to the readme.md
  Info:
    - @garrotte helped me a lot with the mod and made the mod complexity selection possible, so thanks to him!
      Also check out his mods: https://mods.factorio.com/user/garrotte !
    - The old graphite processing will be discontinued in the next update!
